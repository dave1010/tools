---
title: Copilot Session Viewer
description: Render Copilot JSON exports like chat transcripts.
category: Developer Tools
---

Upload a Copilot debug JSON export and browse the conversation like a chat transcript.

Features:

- Upload a Copilot session JSON file and view exportedAt, prompt count, and log count.
- See each prompt, assistant reply, and tool call in order with clear labels and timestamps.
- Expand tool call arguments and responses without leaving the page.

Usage:

1. Export the session from the Copilot debug view in VS Code.
2. Drop or choose the JSON file.
3. Browse the rendered transcript and toggle tool payloads as needed.
4. Download the transcript as Markdown, with or without tool call details, for sharing.
