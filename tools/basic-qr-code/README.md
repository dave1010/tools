---
title: Basic QR Code Generator
description: Generate QR codes from text
category: Data & File Conversion
---

Generate, download, or copy a padded QR code from text and choose its error correction level. The current QR code is saved in the page URL, so you can copy the URL to share it. Shared links use the `data` parameter and optionally the `errorCorrection` parameter (`L`, `M`, `Q`, or `H`).
