Scratch Pad helper

GitHub File to Scratch Pad

Paste a GitHub file link to load its contents directly into the Scratch Pad.

Links like github.com/owner/repo/blob/branch/path/to/file are automatically converted to raw content.

What happens?

  • We convert the link to the raw GitHub user-content URL if needed.
  • The file is fetched and saved to the Scratch Pad's local storage key.
  • Open the Scratch Pad to use the imported text.
