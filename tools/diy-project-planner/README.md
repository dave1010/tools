A simple way to map out a DIY build or repair. Add steps with estimated durations, list your materials with costs, and watch the totals update instantly.

## How to use
1. Name the project and tweak the default steps/materials to match your build.
2. Add more steps with time estimates so the total hours are realistic.
3. Enter material quantities and costs to track budget and remaining items.
4. Use the prep checklist and readiness meter to spot gaps before you start.
